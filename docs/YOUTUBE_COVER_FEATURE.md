# YouTube 视频封面自动提取功能

## 🎯 功能说明

系统会自动从**最后一个视频**（通常是美女视频）提取第一帧作为封面图片。

---

## 📋 工作流程

### 1. 点赞顺序
```
视频1（搞笑）
视频2（搞笑）
视频3（搞笑）
视频4（搞笑）
视频5（搞笑）
视频6（美女）← 最后点赞
```

### 2. 合并顺序
```
播放顺序：视频1 → 视频2 → 视频3 → 视频4 → 视频5 → 视频6
```
- ✅ 按点赞顺序播放
- ✅ 最后点赞的美女视频在最后

### 3. 封面提取
- 📸 自动提取**视频6（美女视频）**的第一帧
- 💾 保存为 `洋了二正#1_cover.jpg`
- 📁 和合并视频在同一目录

## 🎬 使用方法

### 自动提取（推荐）
运行完整流程时自动提取：
```bash
python main.py --youtube
```

系统会：
1. ✅ 下载新点赞的视频
2. ✅ 按点赞顺序合并
3. ✅ 自动提取最后一个视频的封面
4. ✅ 上传到 B 站

### 上传时使用封面
上传 B 站时：
1. 系统会打开上传页面
2. 你可以手动上传 `洋了二正#1_cover.jpg` 作为封面
3. 或者让 B 站自动选择（但可能不理想）

## 🔧 技术细节

### 封面提取命令
```bash
ffmpeg -i 最后一个视频.mp4 -ss 00:00:00 -vframes 1 -q:v 2 封面.jpg
```

### 参数说明
- `-ss 00:00:00` - 从第0秒开始
- `-vframes 1` - 只提取1帧
- `-q:v 2` - 高质量（1-31，数字越小质量越高）

### 封面质量
- 格式：JPG
- 质量：高质量（q:v 2）
- 大小：通常 100-500 KB
- 分辨率：和原视频一致
